# ArcadeIQ Frontend

React + TypeScript web app prototype for ArcadeIQ.

## Run

```powershell
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Purpose

This is the future web surface for ArcadeIQ. It currently uses mock catalog data, but the structure is ready to connect to a backend API for:

- natural-language game search
- AI review summaries
- developer analytics
- collection-based saved games
- collection intelligence summaries
- catalog and purchase data
